Snapspot Framework

A new, minimal CSS framework built with ultra-fast page load times in mind. A must have framework for any website!

View project on GitHub

What is Snapspot Framework?

Snapspot Framework is an upcoming collection of CSS and JavaScript libraries free to use! They make complex tasks like responsive grids and dynamic forms simpler, with some features only needing a few lines of code! The best part about the framework is that it is completely customisable and minimalised. This means that you can pick which module files you want and which ones you don't want. See the 'How to Install' chapter below!

What does it do?

The Snapspot Framework is a collection of libraries that all have import features to help improve the quality of your web pages and to speed up the process of creating them. Below, you can find a list of all the modules that are a part of the framework!

  • Core - This is the core module for the framework and is required if you are to use any other modules.
  • Reset - This module is also required for any of the modules to work. This resets all values to zero and helps keep your web page clean!
  • Grid - This module allows you to create dynamic and responsive grids easily!
  • Typography - This module allows you to easily change the way that the text on your page looks. This formats your headers, paragraphs, lists, etc. to look sleek and clean!
  • Form - This module allows you to format forms and the inputs within them. It allows you to create dynamic and responsive forms that look great and are simple to use!
  • Layout - This module allows you to create responsive layouts on your page, such as a page that includes a header, a footer, some content and a sidebar!

All of the modules in the Snapspot Framework are designed for all devices, and work with any screen size!
